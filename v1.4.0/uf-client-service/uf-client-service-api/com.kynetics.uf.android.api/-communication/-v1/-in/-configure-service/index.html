Class use to build a message to configure the service. When the new configuration can be applied live, a UFServiceMessageV1.Event.ConfigurationUpdated is notified. When the new configuration required a service reboot a UFServiceMessageV1.Event.Stopped and UFServiceMessageV1.Event.Started messages are notified
